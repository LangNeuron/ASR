# Security Policy

## Reporting a Vulnerability
If you discover a security issue, please report it privately through one of these channels:
- Project email: `langneuron@gmail.com`
- Personal email: `anton1programmist@gmail.com`
- Telegram: `@MlSciencePython`

Please include:
- a clear description of the issue;
- reproducible steps or proof of concept;
- potential impact and affected components.

## Response Expectations
- Initial response target: within 72 hours.
- Status updates: provided during triage and remediation.
- Coordinated disclosure is preferred after a fix is available.
