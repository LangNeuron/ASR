# ASR Project

## Название проекта
ASR Project - рабочее пространство для создания собственной модели автоматического распознавания речи (ASR) и сопутствующих аудио-ML инструментов.

## Краткое описание
Репозиторий дает основу для полного цикла разработки речевых ML-систем:
- подготовка и версионирование датасетов;
- извлечение признаков и обучение моделей;
- оценка качества (WER/CER и дополнительные проверки);
- воспроизводимые эксперименты и подготовка к инференсу.

## Структура проекта
- `src/asr/` - исходный код (`data`, `features`, `models`, `training`, `inference`, `utils`).
- `data/` - локальные данные (`raw`, `interim`, `processed`, `external`).
- `notebooks/` - исследовательские и экспериментальные ноутбуки.
- `configs/` - конфигурации экспериментов и пайплайнов.
- `scripts/` - CLI и служебные скрипты автоматизации.
- `models/`, `checkpoints/`, `artifacts/`, `logs/` - результаты обучения и артефакты.
- `tests/` - unit и integration тесты.
- `docs/` - документация проекта и кода на нескольких языках.

## Документация
Документация разделена по языкам и назначению.

Поддержка языков:
- `docs/en/` - документация на английском.
- `docs/ru/` - документация на русском.

В каждой языковой директории одинаковая структура:
- `research-reports/` - отчеты по экспериментам и исследованиям.
- `code-documentation/` - техническая документация по коду и API.
- `theoretical-research/` - теоретические заметки, обзоры и материалы.

## Установка
Требования:
- Python 3.12+
- Poetry

Установка зависимостей проекта:

```bash
poetry install --with dev
```

Установка ML-стека при необходимости:

```bash
poetry install --with dev,ml
```

## Использование
Запуск проверок качества:

```bash
poetry run ruff check .
poetry run mypy src tests
poetry run pytest
```

Запуск ноутбуков:

```bash
poetry run jupyter lab
```

## Pre-commit
Настроены хуки `ruff`, `mypy` и `pytest`.

Инициализация:

```bash
poetry run pre-commit install
poetry run pre-commit run --all-files
```

## Тестирование
Запуск всех тестов:

```bash
poetry run pytest
```

Запуск только интеграционных тестов:

```bash
poetry run pytest -m integration
```
